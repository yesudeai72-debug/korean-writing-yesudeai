# 문장을 만들어요

휴대폰 화면을 우선으로 설계한 한국어 문장 구성 연습 도구입니다.

## 폴더 구성

- `index.html`: 메인 화면
- `styles.css`: 공통 모바일 스타일
- `app.js`: 급수별 샘플 데이터와 상호작용
- `beginner/index.html`: 초급 화면
- `intermediate/index.html`: 중급 화면
- `advanced/index.html`: 고급 화면

## 메모

- 현재 서술어 20개 목록은 구조 확인용 임시 샘플입니다.
- 국립국어원 기초어휘 엑셀 파일을 받으면 급수와 빈도 기준으로 교체할 수 있습니다.
